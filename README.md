# 🚀 NFT Landing Page with Docker

A modern responsive NFT Landing Page project containerized using Docker for easy deployment on any system (Windows, Linux, EC2).

---

## 📌 Project Overview

This project is a simple NFT landing page built using:

- HTML
- CSS
- JavaScript

It is fully containerized using Docker so you can run it anywhere without worrying about dependencies.

---

## ⚙️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Docker
- Nginx (inside container)

---
